About This Project

SkinCare Analyzer adalah aplikasi berbasis web yang memanfaatkan Deep Learning untuk menganalisis kondisi kulit wajah dan memberikan rekomendasi perawatan yang tepat.

🎯 Tujuan Proyek

Membantu pengguna memahami kondisi kulit wajah mereka secara praktis dan informatif melalui teknologi kecerdasan buatan, tanpa harus konsultasi langsung ke dokter untuk pengecekan awal.

  • Klasifikasi kondisi kulit: Normal, Mild, Moderate, Severe Acne
  • Memberikan rekomendasi perawatan berdasarkan hasil analisis
  • Menerapkan Deep Learning pada bidang kesehatan kulit
🗃️ Dataset

Proyek ini menggunakan dataset ACNE Severity Classification (ACNE04) yang bersumber dari Kaggle.

  • Sumber: Kaggle — lexuanhieu131297
  • Format label: IGA Scale (0–3)
  • Kelas: Normal, Mild Acne, Moderate Acne, Severe Acne
  • Split: Train / Validation / Test
⚙️ Teknologi yang Digunakan
Python TensorFlow / Keras MobileNetV2 OpenCV Flask HTML CSS JavaScript Bootstrap 5 GitHub Vercel
🤖 Model yang Digunakan

Model klasifikasi dibangun menggunakan arsitektur MobileNetV2 dengan pendekatan transfer learning. Model di-fine-tune menggunakan dataset ACNE04 untuk mengklasifikasikan kondisi kulit ke dalam 4 kelas.
