About This Project

Built with Care

SkinCare Analyzer menggunakan Deep Learning untuk menganalisis kondisi kulit wajah dan memberikan rekomendasi perawatan yang tepat — praktis, cepat, dan informatif.

🎯 Tujuan Proyek

Membantu pengguna memahami kondisi kulit wajah secara mandiri melalui kecerdasan buatan, tanpa harus konsultasi langsung untuk pengecekan awal.

  • Klasifikasi: Normal, Mild, Moderate, Severe Acne
  • Rekomendasi perawatan berbasis hasil analisis
  • Penerapan Deep Learning di bidang kesehatan kulit

🗃️ Dataset

Menggunakan dataset ACNE Severity Classification (ACNE04) dari Kaggle yang telah melalui proses preprocessing dan augmentasi data.

  • Sumber: Kaggle — lexuanhieu131297
  • Format label: IGA Scale (0–3)
  • Kelas: Normal, Mild Acne, Moderate Acne, Severe Acne
  • Split: Train / Validation / Test

⚙️ Teknologi yang Digunakan

Python TensorFlow / Keras MobileNetV2 OpenCV Flask HTML & CSS Tailwind CSS JavaScript GitHub Vercel

🤖 Model yang Digunakan

Model klasifikasi dibangun menggunakan arsitektur MobileNetV2 dengan pendekatan transfer learning. Model di-fine-tune menggunakan dataset ACNE04 untuk mengklasifikasikan kondisi kulit ke dalam 4 kelas dengan akurasi yang optimal.
