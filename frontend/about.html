About This Project

Built with Care

SkinCare Analyzer menggunakan Deep Learning untuk menganalisis kondisi kulit wajah dan memberikan rekomendasi perawatan yang tepat — praktis, cepat, dan informatif.

🎯 Tujuan Proyek

Mengembangkan sistem berbasis Deep Learning untuk menganalisis kondisi kulit wajah secara otomatis dan memberikan gambaran awal kepada pengguna tanpa harus konsultasi langsung.

  • Klasifikasi kondisi acne pada wajah
  • Analisis tingkat oiliness (kadar minyak kulit)
  • Memberikan hasil analisis yang cepat dan informatif
  • Penerapan AI di bidang skin health monitoring

🗃️ Dataset

Menggunakan kombinasi dataset untuk analisis kondisi kulit wajah, yaitu ACNE Severity Classification (ACNE04) dan Skin Issues Version 2 (Balanced Dataset) untuk analisis oiliness dan kondisi kulit lainnya.

  • Acne: ACNE Severity (IGA Scale 0–3)
  • Oiliness: Indikasi tingkat oiliness berdasarkan intensitas pantulan cahaya pada permukaan wajah
  • Split: Train / Validation / Test

⚙️ Teknologi yang Digunakan

Python TensorFlow / Keras MobileNetV2 Resnet50 Flask HTML & CSS Tailwind CSS JavaScript GitHub Vercel

🤖 Model yang Digunakan

SkinCare Analyzer menggunakan pendekatan transfer learning dengan dua arsitektur deep learning, yaitu MobileNetV2 dan ResNet50, untuk analisis kondisi kulit wajah.

  • MobileNetV2: model ringan untuk inference cepat pada web
  • ResNet50: model dengan ekstraksi fitur yang lebih kuat
